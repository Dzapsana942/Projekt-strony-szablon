# Projekt strony szablon — wielojęzyczny szablon e-commerce

Szablon strony produktowej dostępny w 5 wersjach językowych: 🇵🇱 PL, 🇳🇱 NL, 🇫🇷 FR, 🇧🇪 BE, 🇨🇿 CZ

## Demo
🔗 [dzapsana942.github.io/Projekt-strony-szablon](https://dzapsana942.github.io/Projekt-strony-szablon)

## Funkcjonalności
- Wybór języka/kraju na stronie głównej
- Strona produktowa ze slajderem zdjęć i akordeonem
- Formularz zamówienia z podsumowaniem i obsługą płatności
- Sekcje: dostawa, zwroty, płatności, dane
- Przycisk kontaktu (WhatsApp, telefon, e-mail)
- Dolny pasek nawigacji
- Responsywny design (mobile/desktop)

## Technologie
HTML, CSS, JavaScript
